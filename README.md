**嘉宾分享倒计时**
一个面向会议现场、分享嘉宾和主持人的大屏倒计时网页工具。页面无需安装依赖，打开即可使用，适合投屏、LED 大屏、会议室显示器等场景。
线上访问地址：
https://rancho-yin.github.io/ai-countdown-jiangsu-2026/

产品定位
本工具用于嘉宾分享环节，帮助嘉宾清晰掌握剩余时间，帮助主持人和会务人员更稳定地控制会议节奏。

相比普通手机秒表或系统计时器，它更适合正式会场：

大字号倒计时，远距离也能看清。
支持常用分享时长，一键切换。
可填写嘉宾姓名和分享主题，投屏更正式。
可隐藏控制区，只保留会场展示画面。
时间到后继续显示超时时长，方便主持人判断节奏。

使用方式
在线使用
直接打开：

https://rancho-yin.github.io/ai-countdown-jiangsu-2026/

本地使用
下载或复制本项目后，直接双击打开 index.html 即可使用。

也可以用本地服务预览：

python3 -m http.server 4173
然后访问：

http://127.0.0.1:4173/
推荐现场流程
打开网页。
填写嘉宾姓名和分享主题。
选择分享时长。
点击“全屏”。
点击“隐藏控制”。
嘉宾开始分享时按空格启动倒计时。
如需暂停，再按空格。
下一位嘉宾开始前按 R 重置。
